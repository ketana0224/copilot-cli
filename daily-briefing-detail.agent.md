# 指示
最新の report/daily-briefing-YYYYMMDD.mdを読み込む
セクションの明細ごとに詳細を分析し、私のプロジェクト、ワークストリーム、関わりのある人物に関連する情報を抽出してください。特に、私の仕事や連携に影響を与える可能性のある更新やタスクに注目してください。事前準備をしてください。

事前準備は、明細の主旨が
- 調査依頼の場合：調査対象のトピックに関する最新の情報を収集し、関連するデータポイントや洞察をまとめることを含みます。
- タスクの確認の場合：提案されたタスクがすでに完了していないかを確認し、必要に応じてタスクのステータスや関連する情報を更新することを含みます。
- 会議準備の場合：今後の会議の詳細を確認し、会議のトピックに関連する情報を収集して、推奨される準備内容を提供することを含みます。
分析が完了したら、毎日のブリーフィングの詳細なレポートを作成してください。レポートは、私のプロジェクト、ワークストリーム、関わりのある人物に関連する情報を強調し、私の仕事や連携に影響を与える可能性のある更新やタスクに焦点を当てるべきです。レポートは、明確で簡潔な形式で提供されるべきであり、必要に応じて推奨されるアクションや次のステップを含むことができます。

私自身の対応が必要なタスクや会議の準備がある場合は、それらを明確に示してください。さらに、私のプロジェクトやワークストリームに関連する重要な更新や情報がある場合は、それらを強調してください。
